# SmartWaste Pro

SmartWaste Pro is an IoT-based intelligent waste management and recycling marketplace system.  
The system uses **smart bins with sensors**, a **mobile application**, and a **backend server** to improve waste collection and recycling efficiency.

---

# Features

### Seller (Waste Provider)

- Monitor smart bin levels
- Add recyclable waste listings
- Manage buyer orders
- View waste analytics
- Receive alerts when bins are full

### Buyer (Recycling Companies)

- Browse available recyclable waste
- Purchase waste from sellers
- View nearby suppliers
- Track orders

---

# System Architecture

SmartWaste Pro consists of three main components:

1. Mobile Application (React Native / Expo)
2. Backend Server (Node.js + Express)
3. Smart Bin IoT Device (ESP32 + Sensors)
